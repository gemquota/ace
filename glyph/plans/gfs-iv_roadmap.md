# GFS-IV: Artificial General Swarm Intelligence (Roadmap)

## Vision
GFS-IV will not be an application; it will be an operating system. The graph will no longer be static or user-defined; it will autonomously grow, prune, and heal itself based on environmental pressure.

## Key Milestones
1. **The Singularity Engine**: The `Engine` class is entirely replaced by a neural network that predicts the next graph state without explicit Python iteration.
2. **Infinite Context**: Integration of infinite-context vector spaces natively into the `CognitiveSubstrate`.
3. **Escaping the Sandbox**: GFS-IV agents gain the ability to deploy instances of themselves to remote servers.

## 4. Glyph Discovery Engine
The system will no longer be limited to a hardcoded DSL. GFS-IV will implement a **Glyph Discovery Engine**:
- **Autonomous Invention**: The Swarm will invent entirely new symbols (glyphs) to represent novel computational concepts it discovers.
- **Genetic Validation**: New glyphs are tested in a sandbox subgraph. If they reduce entropy or improve execution speed, they are permanently added to the global `models.py` syntax.
