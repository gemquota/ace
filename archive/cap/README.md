# 🛡️ CAP // COGNITIVE_ARCHITECTURE_PLATFORM

C.A.P is a unified autonomous agent framework built on three pillars:

- **CLIDE** (Command Line Interface Database): The persistent memory and syscall interface.
- **APC Cannon** (Automated Personalized Context and Normalized Neural Ontology Network): The high-speed execution and context optimization engine.
- **PIE** (Praxis Inference Engine): The adaptive reasoning and causal inference core.

## 📂 Project Structure

- `core/`: The heart of the system.
    - `clide/`: Database, Kernel, Storage, Swarm management, and Dashboard.
    - `apc/`: Execution, Sandboxing, and Context management.
    - `pie/`: Inference, Causal modeling, and Logic engines.
- `data/`: Persistent storage (SQLite databases, JSON models).
- `docs/`: Technical documentation and UI mockups.
- `logs/`: System logs and trace histories.
- `scripts/`: Utility scripts for maintenance and reporting.

## 📟 Dashboard

The CAP Dashboard provides real-time observability into swarm activities, intent markets, and cognitive telemetry.

**Access:** Run `python core/clide/dashboard/server.py` and navigate to `http://localhost:8080`.
